# Coming Soon Landing Page

A clean and simple **Coming Soon landing page** built using **HTML and CSS**.  
This project can be used as a temporary placeholder while a website or product is under development.

## Features

- Clean and minimal UI
- Responsive page layout
- Email input section for notifications
- Image integration
- Easy to customize

## Technologies Used

- HTML5
- CSS3

## How to Use

1. Download or clone the repository
2. Open the `coming_soon.html` file in any web browser
3. Customize text, images, or styles as needed
